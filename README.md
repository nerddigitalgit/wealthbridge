# WealthBridge Website

Next.js website with Builder.io visual CMS integration.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **CMS:** Builder.io
- **Deployment:** Vercel
- **Styling:** CSS with CSS Variables

## Environment Variables

Required environment variables in `.env.local`:

```
NEXT_PUBLIC_BUILDER_API_KEY=your_api_key_here
```

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment

Deployed on Vercel from the `nextjs-builderio` branch.

Live site: https://wealthbridge-v2-import.vercel.app/

## Visual Editing

Use Builder.io dashboard to visually edit content, colors, images, and layout without touching code.
